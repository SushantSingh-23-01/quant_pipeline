# 📉 Trading Strategy Sandbox

A personal repository for dumping, testing and archiving trading strategies found online.

The goal isn't to build a flawless commercial system, but to quickly write wrapper pipelines using `backtesting.py` and `pandas_ta` to see if these 
internet strategies actually survive real-world market data. Spoiler: Most don't!

## Current Setup
- `bactesting.py` - For running even-driven backtests.
- `pandas_ta` - For quick technical analysis indicator mapping.
- `yfinance` - For downloading historical market data.
